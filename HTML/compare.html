What file formats does this tool support?

This tool can read Excel files (XLS/XLSX/XLSM/XLSB)

Does this tool compare against formulas?

No, this tool does value based comparisons for excel files, two files are compared per rows/column wise.

How It Works

  • You can drag and drop your files in the upload space or click on the browse file to load files directly from local storage.
  • After both files are uploaded, click on the compare button to compare your excel files.
  • When comparison is completed,series of options will be made for use.
